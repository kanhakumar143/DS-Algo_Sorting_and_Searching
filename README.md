# DS-Algo_Sorting_and_Searching

## Time Complexity Of Searching Algorithm

1. **Bubble Sort**
   - ***Time Complexity*** **O(n^2)**


2. **Insertion Sort**
   - ***Time Complexity*** **O(n^2)**


3. **Selection Sort**
   - ***Time Complexity*** **O(n^2)**


4. **Quick Sort**
   - ***Space Complexity*** **O(n)**
   - ***Time Complexity*** 
   - **Best Case** **O(nlogn)**
   - **Average Case** **O(nlogn)**
   - **Worst Case** **O(n^2)+O(nlogn)**

5. **Merge Sort**
   - ***Time Complexity*** **O(nlogn)**
